# Bible Battle - 競技聖書バトル 実装計画

## コンセプト
2人の牧師がディベートするように、プレイヤーとAIが交互に聖書の聖句を引用して対決するWebゲーム。
ラウンドごとにテーマが出題され、そのテーマに最も関連する聖句を引用した方が高スコアを獲得する。
聖句が出されるたびに、AI実況者がバトルを盛り上げる解説コメントを生成する。

## 技術スタック
- **フロントエンド**: React + TypeScript + Vite
- **スタイリング**: Tailwind CSS
- **聖書データ**: 静的JSON（英語版KJVから開始、後に和訳データ差し替え）
- **スコアリング**: キーワードマッチベース
- **AI実況**: テンプレート + ロジックベース（外部API不要）

## Phase 1: データ準備

### 1.1 聖書データ取得・整形
- Kaggleから英語聖書データ（KJV）をダウンロード
- パースして上記JSON構造に変換するスクリプト (`scripts/prepare-bible-data.ts`)
- 66書、1,189章、31,102節を格納

### 1.2 テーマデータ作成
- 10〜15テーマを手動で定義
- 各テーマに関連キーワードリスト（英語）を付与
- テーマ例: 愛、信仰、赦し、苦難、知恵、正義、希望、祈り、創造、救い

## Phase 2: コアゲームロジック

### 2.1 聖句検索エンジン (`src/lib/bible.ts`)
- 入力パーサー: `"John 3:16"` → `{ book: "John", chapter: 3, verse: 16 }`
- 書名のエイリアス対応: `"1 Cor"`, `"1 Corinthians"`, `"I Corinthians"` → 同じ書
- 範囲指定: `"Romans 8:28-30"` → 複数節取得（将来対応）
- バリデーション: 存在しない章/節のエラーハンドリング

### 2.2 スコアリングシステム (`src/lib/scoring.ts`)
- テーマのキーワードリストと聖句テキストを照合
- スコア計算:
  - キーワード1つヒット = 10点
  - ユニークキーワード数ボーナス（複数種類ヒットで加点）
  - 聖句の長さによる正規化（長い聖句が有利にならないよう）
- 0〜100点のスケールに正規化

### 2.3 AI対戦相手 (`src/lib/ai-opponent.ts`)
- テーマごとに関連度の高い聖句をあらかじめインデックス化
- 難易度調整:
  - Easy: ランダムに近い選択
  - Normal: 上位50%から選択
  - Hard: 最適な聖句を選択
- 同じ聖句を繰り返さないようにする

### 2.4 AI実況システム (`src/lib/commentary.ts`)
- テンプレートベースの実況コメント生成
- コメントパターン:
  - 聖句引用時: 「おおっと！{player}が{book} {chapter}:{verse}を繰り出した！」
  - 高スコア時: 「これは見事な選択！テーマ'{theme}'にド直球です！」
  - 低スコア時: 「うーん、少しテーマから外れたか...？」
  - 逆転時: 「ここで逆転！{player}が一気にリードを奪った！」
  - 接戦時: 「両者譲らない白熱のバトル！」
- スコア差やラウンド状況に応じてコメントを動的に選択
